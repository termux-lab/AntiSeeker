# AntiSeeker
Utility for crashed Seeker
# Install and start
git clone https://github.com/termux-lab/AntiSeeker.git<br>
cd AntiSeeker<br>
python3 antiseeker.py -h<br>
-h    Help<br>
-p    Parsing the Seeker<br>
-c    To crash Seeker<br>
# Example
python3 antiseeker.py -p https://******.ngrok.io<hr>
