# AntiSeeker
Utility for crashed Seeker
# Install and start
git clone https://github.com/termux-lab/AntiSeeker.git<br>
cd AntiSeeker<br>
python3 antiseeker.py -h<br>
```
-h    Help
-p    Parsing the Seeker
-c    To crash Seeker
```
